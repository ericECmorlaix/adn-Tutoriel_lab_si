Démarrer le Rpi en l'alimentant **uniquement avec l'alimentation 7,5V** via la carte pixl du Poppy Ergo Jr.

Raccorder le Poppy Ergo Jr à votre LAN.

Saisir dans un navigateur l'adresse IP du Poppy (par exemple http://192.168.5.102/) ou saisir http://poppy-1.local/ ou http://poppy-2.local/ selon le nom du robot.
